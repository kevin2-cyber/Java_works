IDE tips & tricks

Learning is a journey with a lot of experiment and investigation. Look under the hood of any code and see what's going on with the help of the IntelliJ IDEA graphical debugger.

Set a line breakpoint

by clicking the left gutter area at a line where you want to stop, or by pressing Cmd + F8:

Now run the debugger with Ctrl + Shift + D:

Your program will stop on the line with breakpoint:

Now use F8 to go step-by-step through your code and find all the information about what is happening in the Debug tool window:

You can also just hover over a code expression and see the evaluation result:
