Command-line arguments

A Java application can accept data from the external world using command-line arguments.

Here is a declaration of the main method:

public static void main(String[] args)

The array of string args contains string arguments from the command line.

Let's assume we have the following code in the file Task.java:

public class Task {
  public static void main(String[] args) {
    System.out.println("passed arguments: " + args.length);
  }
}

This code outputs the string containing a number of passed arguments.

Let's compile this code (the result is bytecode):

javac Task.java

Then run the bytecode without parameters:

java Task

It outputs:

passed arguments: 0

Let's run it with two parameters separated by space:

java Task arg1 arg2

It outputs:

passed arguments: 2

So, we have learned that in the command-line interface, arguments must be separated by spaces.

IDE tips & tricks
