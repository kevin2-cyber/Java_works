IDE tips & tricks

Smart autocompletion is an essential feature of a professional code editor, which helps you to save time and avoid misprints. It is equally powerful for learning because you can use it to find the right variable, function, parameter, or code construct. Start typing or simply press Ctrl + Space to get a list of suggested options:

Code snippets, or live templates, can also be a handy tool. They not only speed up your coding, but help you learn new language syntax as well. Use Cmd + J to see the full list of code snippets available. Start typing to find the one you want to add, and then use Tab to fill in the blanks:
