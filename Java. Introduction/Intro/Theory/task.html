Getting started with the course

This is an introductory course to Java. The course covers the basic concepts used to get going with the language. Each lesson may include theory, quizzes, and programming practice assignments. Read all the theory, check your understanding answering the quiz questions and practice with programming assignments.

Note: If you like learning the theory lessons on Stepik but prefer to do the coding exercises on IntelliJ IDEA - you don't need to worry about losing your course progress. All your progress is synchronized between the platforms. Follow IDE tips & tricks section helping you to be more productive while learning with IntelliJ IDEA.

IDE tips & tricks

When you open your course, you will see the main tool windows used to get around: Project View, Editor, Task Description.

Project View

Project View shows the course syllabus: list of lessons with lesson steps, or tasks. To navigate to a task, click on its name.

Note: You can hide the Project View window by clicking on the Project tool button or with the ⌘1 shortcut. This will give you more space for code and the task description. Click the Project tool button (or invoke ⌘1) once again to show the hidden Project View.

Task Description

The Task Description window gives you all the information you need to complete a task:

  • Learning and reading materials for theory tasks
  • Question and variants of the answers to the quizzes
  • Problem descriptions for programming assignments

Use Task Description icons for the following actions:

  •  – to check if a task assignment is done right for the quizzes and programming tasks (⌥⌘⏎)

  •  – to run code from the editor for the theory tasks (⌥⌘⏎)

  •  – to go to the previous/next task (⌘,/⌘.)

  •  – to drop all the task changes and start it anew (⇧⌘X)

Task Description needs to be visible, to hide it completely is not advised. But still, you can make it a bit less distracting by moving it out of the way.

Editor

The Editor is your playground and where you will mainly be programming. You can experiment here for the theory tasks and quizzes without being checked. For programming assignments in the Editor you will find the code you need to fix, or you will write your own code from scratch. This code will be checked.

You can always run your code with the icon on the left gutter

.

Note: If you want to easily go back to the Editor and focus on your code, the Hide All Windows command (⇧⌘F12) is the best option. Just invoke it once again to get all the windows back.
